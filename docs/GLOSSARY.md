# Glossary

## Terms

**STX**: Stacks native token

**microSTX**: Smallest unit of STX (1 STX = 1,000,000 microSTX)

**Lock Period**: Duration STX is locked in blocks

**Reputation Points**: Points earned on successful withdrawal

**Badge**: NFT achievement token

**SIP-009**: Stacks NFT standard

**Streak**: Consecutive deposits

**Time Multiplier**: Bonus for longer lock periods

**Early Withdrawal**: Withdrawing before maturity

**Penalty**: Fee for early withdrawal (20%)
