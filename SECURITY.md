# Security Policy

## Reporting Security Issues

If you discover a security vulnerability, please email: security@bitsave.example

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Security Features

- Reentrancy protection
- Input validation
- Overflow protection
- Admin rate limiting
